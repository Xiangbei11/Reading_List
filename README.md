“读书之于我，那是人生之中最重要事，只次于呼吸空气、饮水、吃饭、睡觉。我曾经想：坐牢十年而可以在狱中阅读天下书籍，或者是十年中充分自由，但不得阅读任何书刊——两者由我选择，我一定选择‘坐牢读书’。”
